# 用例分析文档

**项目：** 中国象棋·开局大师
**日期：** 2026-04-13
**类型：** 概要级用例分析

---

## 参与者

| 参与者 | 描述 |
|--------|------|
| **普通用户** | 可使用免费开局 |
| **付费用户** | 可使用全部开局（免费+付费解锁） |
| **AI引擎** | 严格按照棋谱回应（非智能AI） |
| **微信支付** | 处理付费开局解锁 |

---

## 用例清单

| 用例 | 参与者 | 描述 |
|------|--------|------|
| **UC1. 浏览开局库** | 普通用户、付费用户 | 查看分类、选择变例、查看逐变讲解 |
| **UC2. 观看教学视频** | 普通用户、付费用户 | 播放/暂停/拖拽视频（可选） |
| **UC3. 开局练习** | 普通用户、付费用户 | 选择先后手、走棋判定、AI回应、悔棋、结算 |
| **UC4. 解锁付费开局** | 付费用户、微信支付 | 微信支付内购解锁付费开局 |

---

## 参与者与用例关系

```
普通用户 ──→ UC1. 浏览开局库
              UC2. 观看教学视频
              UC3. 开局练习
              UC4. 解锁付费开局（成为付费用户）

付费用户 ──→ UC1. 浏览开局库（全集）
              UC2. 观看教学视频
              UC3. 开局练习
              UC4. 解锁付费开局

AI引擎 ──→ UC3. 开局练习（棋谱回应子流程）

微信支付 ──→ UC4. 解锁付费开局
```

---

## 付费用户与普通用户的区别

| 区别项 | 普通用户 | 付费用户 |
|--------|----------|----------|
| 开局库范围 | 仅免费开局 | 全部开局（免费+付费解锁） |
| 教学视频 | 有 | 有 |
| 逐变讲解 | 有 | 有 |
| 悔棋功能 | 有 | 有 |
| 结算功能 | 有 | 有 |

**结论：** 付费用户与普通用户的唯一区别是开局库范围，其他功能完全相同。
