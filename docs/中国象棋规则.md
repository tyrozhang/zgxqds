# 中国象棋规则

## 一、棋盘与棋子

### 1.1 棋盘结构

中国象棋棋盘由9条竖线和10条横线交叉组成，形成90个交叉点。

- **九宫**：棋盘两端各有一个3×3的方格区域，用斜线表示，是将帅活动的区域
- **楚河汉界**：棋盘中间的空白地带，分隔双方阵地
- **河界**：棋盘中间第5、6条横线之间为河界，部分棋子过河后才能横向移动

### 1.2 棋子分类

每方共有16枚棋子：

| 棋子 | 数量 | 颜色 |
|------|------|------|
| 将/帅 | 1 | 黑方将/红方帅 |
| 士/仕 | 2 | 黑方士/红方仕 |
| 象/相 | 2 | 黑方象/红方相 |
| 马/傌 | 2 | 黑方马/红方傌 |
| 车/俥 | 2 | 黑方车/红方俥 |
| 炮/砲 | 2 | 黑方炮/红方砲 |
| 卒/兵 | 5 | 黑方卒/红方兵 |

### 1.3 初始摆盘

黑方底线（第10横线）：从左到右依次为车1、马2、象3、士4、将5、士6、象7、马8、车9
黑方第9横线：炮在2路和8路位置
黑方第7横线：卒在1、3、5、7、9路位置

红方第4横线：兵在1、3、5、7、9路位置
红方第2横线：砲在2路和8路位置
红方底线（第1横线）：从左到右依次为俥1、傌2、相3、仕4、帅5、仕6、相7、傌8、俥9

## 二、棋子走法

### 2.1 将/帅

**活动范围**：只能在九宫内移动（本方底线、次底线、第三条横线，以及4、5、6路竖线围成的区域）

**移动方式**：横、竖各走一格，可上下左右移动

**移动点数**：在九宫中心最多有4个可走位置，在四角只有2个可走位置，在边线中心有3个可走位置

**特殊规则**：将帅不能在同一直线上直接对面（中间无棋子阻隔），此规则称为"将帅不见面"或"白脸将"

### 2.2 士/仕

**活动范围**：只能在九宫内移动

**移动方式**：斜走一格，即沿九宫对角线移动，只能走四个斜线方向

**移动点数**：在九宫四角各有1个可走位置（只能走向中心），在中心位置有4个可走位置（可走向四角）

**功能**：专用于保护将/帅，是九宫内唯一的斜线移动棋子

### 2.3 象/相

**活动范围**：不能过河，只能在本方阵地活动。红方相可在第1、3、5横线（不超过河界），黑方象可在第6、8、10横线（不超过河界），竖线均为1-9路

**移动方式**：走田字格，即从田字一角走到对角，横向移动两格且纵向移动两格

**塞象眼（蹩象眼）**：田字中心点有任意棋子时，象不能从一角走到对角。例如黑象在(7,10)要去(5,8)，若(6,9)有棋子，则不能移动；又如红相在(3,3)想去(5,1)，若(4,2)有子则塞象眼

**移动点数**：每方两个象各自控制7个点位，全盘共14个点位（但象之间不能相互保护，需根据具体位置判断）

### 2.4 马/傌

**活动范围**：全盘任意位置均可到达

**移动方式**：走日字格，先直线走一格，再斜走一格（走法类似"L"形，共8个可能方向）

**蹩马腿**：马在前进方向的相邻点（直走一格的位置）有任意棋子时，该方向的日字移动被阻挡。例如马在(3,10)要去(4,8)，若(3,9)有棋子，则该方向不能移动；但去(2,8)不受(3,9)影响

**移动点数**：马在棋盘中心最多有8个可走位置，在边线减少，在角落只有2个可走位置

### 2.5 车/俥

**活动范围**：全盘任意位置均可到达

**移动方式**：横、竖任意格数，沿直线行走，只要路径上无阻碍，可走到同一直线上的任意空位

**吃子规则**：行走路径终点是对方棋子时可吃掉该子，但中间不能有任何棋子阻隔（包括己方和对方棋子）

**移动点数**：车在空旷棋盘上控制17个点位（同横线8个+同竖线9个-自身位置），是攻击力最强的棋子

### 2.6 炮/砲

**活动范围**：全盘任意位置均可到达

**移动方式**：横、竖任意格数，沿直线行走，与车的移动方式完全相同。只要路径上无阻碍，可走到同一直线上的任意空位

**吃子规则（炮打隔山）**：吃子时必须隔一个棋子（称为"炮架"或"炮台"），即路径上有且只有一个棋子（可以是己方或对方棋子），且目标位置是对方棋子，才能吃掉。移动时不能吃子，吃子时不能单纯移动

**移动点数**：炮在空旷棋盘上控制17个点位，但攻击范围取决于炮架位置

### 2.7 卒/兵

**活动范围**：全盘，但移动方式受过河限制

**未过河时**：只能向敌方方向直线前进一格（红方兵向上，即横线号增加；黑方卒向下，即横线号减少）

**已过河时**：可以向敌方方向前进一格，或向左右横向移动一格，共3个可能方向（边线卒只有2个方向）

**过河判定**：红方兵过河指从第5横线进入第6横线及以上；黑方卒过河指从第6横线进入第5横线及以下

**限制**：卒/兵永远不能后退，即不能向本方方向移动

**移动点数**：未过河时只有1个方向，过河后最多3个方向，到底线后只能横向移动

## 三、基本规则

### 3.1 将军与应将

**将军定义**：一方的将/帅被对方棋子的下一步可以吃掉的状态。将军时必须声明"将军"或"照将"

**将军方式**：
- 直接攻击将/帅所在位置
- 控制将/帅要走的位置（困毙前的将军）
- 通过照面规则迫使对方无法应将（白脸将）

**应将方法**：被将军的一方必须在下一步解除将军状态，主要有三种应将方式：

1. **吃子解将**：吃掉对方进行将军的棋子。适用于对方将军棋子可以被本方棋子攻击到的情况

2. **垫将（阻挡）**：在对方攻击线路上放置一个棋子进行阻挡。适用于对方车、炮、卒/兵的直线攻击，以及将/帅之间的照面

3. **避将（移动）**：将/帅移动到不被攻击的安全位置。适用于将/帅周围有空位且不被攻击的情况

**应将失败**：若以上三种方法均无法解除将军状态，则为将死，对局结束

### 3.2 胜负判定

**将死**：一方被将军且无法应将（无法吃子、无法垫将、无法避将），对方获胜。将死是正常对局的主要结束方式

**困毙（欠行）**：一方未被将军，但所有棋子均无合法棋可走（所有移动都会导致被将军或违反规则），对方获胜。困毙通常发生在残局阶段

**认输**：一方主动认输，对方获胜。通常在局面大劣、无法挽救时选择

**超时**：比赛用时耗尽，对方获胜。正式比赛有计时规则，超时判负

**违规判负**：严重违反比赛规则（如两次长将不变招），对方获胜

### 3.3 和棋判定

**理论自然和棋**：双方子力不足以构成杀局，按照象棋规则无法取胜的局面。例如：
- 单象对单士
- 单象对单象（双方象为同侧）
- 单士对单士
- 单车对单士象（正确防守下）
- 双士对单兵（兵未过河）

**长将（禁止着法）**：一方连续不断地将军，但无法将死对方；另一方也无法摆脱被将军的循环。单方长将属于禁止着法，长将方必须变招，不变招者判负。防守方可在长将循环中要求和棋

**长捉（禁止着法）**：连续捉对方同一枚无根子（没有其他棋子保护的棋子）形成循环，需变招，不变判负。双方互捉形成循环属于长捉类，亦须变招

**自然限着（60步规则）**：连续60回合（双方各走60步，共120着）内双方均未吃掉对方任何棋子，任一方可提出自然限着。若防守方未能以和棋规定应将，经审核后由裁判裁定胜负

**三次重复局面**：同一局面出现三次，轮到走棋的一方可提出和棋

**提议和棋**：任一方在己方走棋时可提出和棋提议，对方同意则判和

## 四、特殊规则

### 4.1 将帅不见面（白脸将）

**规则内容**：双方将帅不能在同一直线上直接相对（中间无任何棋子阻隔）。若一方将/帅移动到与对方将/帅同一竖线且中间无子，属于非法走法

**应用**：
- 若将/帅照面，主动造成照面的一方判负（除非对方立即应将）
- 此规则常用于残局制造杀法，如"白脸将杀"
- 一方的将/帅可限制对方将/帅的活动范围

**示例杀法**：己方车或兵控制中路，迫使对方将/帅移动，己方将/帅跟进形成照面，用车或兵完成绝杀

### 4.2 长将限制

**长将定义**：一方连续不断地将军，形成循环着法（每两步循环一次或每四步循环一次等）

**规则**：
- 单方长将属于禁止着法，长将方必须变招
- 若长将方不变招，经三次循环后判负
- 防守方可在长将循环中要求和棋

**允许情况**：
- 一将一杀（将军后走一步杀棋，再将军）：属于允许着法，因意图杀棋
- 一将一捉：通常允许，但需具体分析
- 双方互将：形成循环，通常判和

### 4.3 禁止着法

**长捉**：连续捉对方同一枚无根子（没有其他棋子保护的棋子）形成循环，需变招，不变判负

**长拦**：连续阻挡对方棋子形成循环，一般允许，除非明显消极

**一将一闲**：将军后走一步闲棋，再将军，循环三次以上。若无法构成杀势，需变招

**一杀一闲**：走一步杀棋（威胁将死），走一步闲棋，循环三次以上，视具体情况判定

**马步循环**：利用马的特殊走法形成捉子循环，需具体分析是否为长捉

**判例原则**：
- 进攻方应积极求胜，不能消极循环
- 防守方允许用循环着法谋求和棋
- 一方长将、长捉，另一方长杀、长兑，前者需变招

## 五、记谱方法

### 5.1 数字记谱法

使用四位数字表示一步棋：
- 第1、2位：起始位置（竖线+横线）
- 第3、4位：目标位置（竖线+横线）

坐标说明：竖线为1-9（从左到右），横线为1-10（红方底线为1，黑方底线为10）

示例：
- 炮二平五（红方二路炮平到五路）：红方砲从(2,2)到(5,2)，记为2252
- 马八进七（红方八路马进到七路）：红方傌从(8,1)到(7,3)，记为8173
- 卒3进1（黑方3路卒向前走一步）：黑方卒从(3,7)到(3,6)，记为3736

### 5.2 中文记谱法

格式：[棋子][位置][动作][目标]

**位置表示**：
- 红方用中文数字一至九表示竖线（从右到左）
- 黑方用阿拉伯数字1-9表示竖线（从右到左）
- 注意：双方视角相反，红方一路是黑方9路

**动作表示**：
- 进：向敌方方向移动（红方横线号增加，黑方横线号减少）
- 退：向己方方向移动（红方横线号减少，黑方横线号增加）
- 平：横向移动，横线不变

**目标表示**：
- 进退：用数字表示到达的横线
- 平：用数字表示到达的竖线

**示例**：
- 炮二平五：红方二路炮横向走到五路（当头炮开局）
- 马八进七：红方八路马向前走到七路位置（横线7）
- 车一平二：红方一路车横向走到二路
- 炮8平5：黑方8路炮横向走到5路（应对当头炮）
- 马2进3：黑方2路马向前走到3路位置（横线8）
- 卒3进1：黑方3路卒向前走一步（从横线7到横线6）

**同线多子处理**：
- 前[棋子]：同线靠近敌方的一子前进
- 后[棋子]：同线靠近己方的一子后退
- 中[棋子]：同线中间位置的一子移动
