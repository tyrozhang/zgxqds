# 中国象棋·开局大师 - 需求规划

## 项目概述

**项目名称：** 中国象棋·开局大师
**项目类型：** 微信小程序
**目标用户：** 中级和高级象棋玩家
**核心定位：** 专注开局布局学习与练习，让用户通过本系统熟记棋谱（不包含残局内容）。

## 需求总结

### 1. 目标用户

- 中级和高级象棋玩家
- 聚焦开局布局能力提升

### 2. 核心功能

| 功能 | 描述 |
 | ------|------| 
| **开局棋谱库** | 提供经典开局参考（中炮、飞相、仙人指路等） |
| **教学视频** | 每个开局配有教学视频，用户可选择观看学习 |
| **练习** | 与固定棋力 AI 对弈开局，熟记棋谱 |
| **逐变讲解** | 展示棋谱分支变化，辅助理解棋路 |

### 3. AI 分析模式

- **本版本去掉AI分析功能**

### 4. 对弈模式

- 仅人机对弈（不需要人人对弈）

### 5. 学习路径

- 自由选择开局类型（不需要闯关或推荐路线）
- **可选观看教学视频** → 或直接开始练习

### 6. 用户系统

- 微信授权登录
- 用户数据：openid + 昵称 + 头像
- 用户设置本地存储（不云端同步）
- **去掉历史对局功能**

### 7. 数据策略

- 所有开局库存储在本地，完全离线可用
- 开局库包含免费开局和付费锁定开局
- 付费开局通过微信支付解锁

## 已明确的需求细节

### 开局练习流程

1. 选择开局体系（如"中炮对屏风马"）
2. 在体系内选择具体变例
3. **进入开局详情页，可选择观看教学视频或直接开始练习**
4. 选择先手/后手（执红/执黑由先后手决定）
5. 进入对弈界面（木质棋盘 + 可交互走子）
6. **严格棋谱模式**：AI 和棋手必须严格按照所选棋谱走棋，AI 不做任何计算
7. 用户走棋 → 系统判断：

   - **非棋谱走法**（不在当前棋谱中）→ 提示"非当前棋谱棋路，请再回去继续学习当前棋谱吧~"，不走棋
   - **典型错误走法**（在棋谱的错误分支中）→ 提示典型错误信息，允许继续下棋
   - **正确棋谱走法** → 正常走棋，AI 按棋谱回应
8. 对弈结束 → 显示结果，返回首页

### 严格棋谱模式规则（已确定）

- 练习时必须严格按照所选棋谱走棋
- AI 不做任何计算，仅按棋谱数据库回应
- 用户走出非棋谱走法时，提示学习信息，不走棋
- 用户走出典型错误走法时，仅提示错误信息，允许继续

### 错误走法处理（已确定）

术语说明：
- **非棋谱走法**：棋手走出了当前所选棋谱中不存在的棋路
- **典型错误走法**：棋手走出了棋谱中标注为典型错误的分支

处理规则：

- **非法走法**：禁止落子，棋子无法移动
- **非棋谱走法**：提示"非当前棋谱棋路，请再回去继续学习当前棋谱吧~"，不走棋，双方保持原局面
- **典型错误走法**：提示预置的典型错误信息（如"此走法会导致失先"），允许继续下棋，AI 按棋谱错误分支回应

### 教学视频（新增）

- 视频存储：云端 CDN
- 加载方式：进入开局详情页时按需加载
- 播放控制：支持播放、暂停、进度拖拽
- 离线状态：无网络时隐藏视频入口，直接显示"开始练习"按钮
- 可选观看：用户可随时跳过视频直接开始练习

### 逐变讲解交互（已确定）

- 点按分支 → 棋盘切换到该分支下一步局面，始终在当前视图
- 分层展开，点击才展开下一层变化
- 单路径树形浏览：只保持一条展开路径，用户可后退到分岔点再选另一分支

### AI 分析模式（已确定）

- **本版本去掉AI分析功能**，专注棋谱练习体验

### 回放功能（已删除）

- **本版本去掉历史对局和回放功能**
- 对弈结束后直接显示结果，用户可选择重新练习或返回首页

### 开局库结构（已确定）

| 层级 | 内容 | 存储方式 |
| ------ | ------ | --------- |
| 开局库 | 所有开局变例 | 本地打包 |
| 锁定库 | 付费锁定开局（用户需购买后使用） | 本地数据，付费标记解锁 |

### 开局定义（已确定）

- 体系仅作为组织分类方式，不作为计数单位
- 上线时由数据录入/导入，开发期不关心具体变例内容

### 交互（已确定）

- 非棋谱走法：提示"非当前棋谱棋路，请再回去继续学习当前棋谱吧~"
- 典型错误走法：提示错误信息，允许继续
- 取消选中：无取消操作

### 首屏加载（已确定）

- 需要 Loading 页

### 数据存储方案（已确定）

- 所有开局库存储在本地
- 无云端下载逻辑
- 完全离线可用

### 悔棋规则（已确定）

- 任何时候均可悔棋
- 不限制悔棋步数
- AI 走完后也能悔棋
- 悔棋是步数回退，无需考虑分支问题

### 对弈规则（已确定）

- 不能同时开多局对弈
- 对弈中途退出：对局丢弃，不保存

### 胜负判定规则（已确定）

- 对弈结束后显示结果，提供"重新练习"和"返回首页"选项
- **去掉回放功能**

### 用户数据模型（已确定）

- 用户数据：openid + 昵称 + 头像
- **去掉历史对局存储**
- 用户设置：本地存储，不云端同步

### 云端数据更新（已确定）

- **本版本无云端数据更新功能**
- 开局库随小程序版本发布更新

### 锁定库机制（已确定）

- **解锁方式：付费解锁** - 用户通过微信支付内购解锁特定开局
